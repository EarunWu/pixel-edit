# Pixel Edit

A lightweight browser-based pixel art editor built with Vite, React, and TypeScript.

## Features

- Pencil, eraser, and eyedropper tools
- Undo and redo history
- 16x16, 32x32, and 64x64 canvas presets
- Pixel grid toggle and zoom controls
- Image import with fit-to-canvas or capped original-size modes
- Transparent PNG export at 1x, 4x, 8x, or 16x scale

## Development

```powershell
npm install
npm run dev
```

## Checks
